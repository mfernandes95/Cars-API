# Cars-api

API in NodeJS using: Express, JWT, Postgres, Jest, Docker, Swagger. To authenticate users and CRUD cars.



## Run

```bash
sudo docker-compose up --build
```

## Get Documentation

http://localhost:3333/api-docs/
